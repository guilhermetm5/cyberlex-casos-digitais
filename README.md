# CyberLex: Casos Digitais

CyberLex: Casos Digitais é um jogo educativo desenvolvido em Godot com o objetivo de ensinar conceitos de Direito Cibernético, segurança digital, proteção de dados e investigação de incidentes por meio de casos interativos.

## Objetivo

O projeto busca transformar conteúdos jurídicos e técnicos em experiências práticas, colocando o jogador no papel de um analista jurídico-digital responsável por investigar situações como phishing, vazamento de dados, uso indevido de imagem, fake news, invasão de contas e outros conflitos digitais.

## Temas abordados

- Direito Cibernético
- Segurança da informação
- Proteção de dados
- LGPD
- Crimes cibernéticos
- Provas digitais
- Ética digital
- Responsabilidade online

## Ideia do jogo

O jogador recebe casos simulados e precisa tomar decisões com base em evidências, normas jurídicas e boas práticas de segurança digital.

Cada decisão pode afetar indicadores como:

- Precisão jurídica
- Qualidade da investigação
- Preservação de provas
- Ética
- Risco para a vítima ou organização

## Status do projeto

Em fase inicial de planejamento e prototipagem.

## Tecnologias previstas

- Godot Engine
- GDScript
- JSON para estruturação dos casos
- Interface 2D interativa

## Casos planejados

1. E-mail de phishing contra funcionário
2. Vazamento de dados de clientes
3. Perfil falso usando imagem de vítima
4. Invasão de conta em rede social
5. Fake news e difamação online
